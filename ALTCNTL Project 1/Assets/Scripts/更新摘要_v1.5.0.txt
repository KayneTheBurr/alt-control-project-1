🎮 新按键布局

  小键盘1  →  立即左倾倒 / 从右倾倒跨越
  小键盘2  →  向左倾斜一个阶段
  小键盘3  →  向右倾斜一个阶段
  小键盘4  →  立即右倾倒 / 从左倾倒跨越

  ⏱️ 长按任意键1秒 → 再次触发

───────────────────────────────────────────────────

⚡ 核心机制

  【长按重复】
    按下 → 立即触发
    持续 → 1秒后再触发
    松开 → 重置计时

  【倾倒跨越】
    左倾倒(-3) + Keypad4 → 跨4阶段 → 右倾1(1)
    右倾倒(3)  + Keypad1 → 跨4阶段 → 左倾1(-1)

  【倾倒回退】（保留）
    左/右倾倒 + Keypad2/3 → 回到对应的倾2阶段

💡 使用技巧

  • 连续操作：按住小键盘2/3可每1秒倾斜一次
  • 紧急救援：倾倒时按相反方向倾倒键快速跨越
  • 回退恢复：倾倒时按小键盘2/3逐级回退
  • 禁用长按：Inspector取消勾选 Enable Hold Repeat
  • 调整触发时间：Inspector修改 Hold Duration
