───────────────────────────────────────────────────

🎯 按键功能

  【1键】立即左倾倒 + 掉落
    • 直接进入左倾倒阶段
    • 立即掉落1个物品
    • 适合紧急情况

  【2键】向左倾斜一个阶段
    • 居中 → 左倾1 → 左倾2 → 左倾倒
    • 逐级移动
    • 有过渡动画

  【3键】向右倾斜一个阶段
    • 居中 → 右倾1 → 右倾2 → 右倾倒
    • 逐级移动
    • 有过渡动画

  【4键】立即右倾倒 + 掉落
    • 直接进入右倾倒阶段
    • 立即掉落1个物品
    • 适合紧急情况

───────────────────────────────────────────────────

💡 使用技巧

  【正常游戏】
    主要使用 2 和 3 键
    逐步调整平衡

  【紧急处理】
    使用 1 或 4 键
    快速丢弃物品

  【战略丢弃】
    堆叠过高时
    主动倾倒减少高度

───────────────────────────────────────────────────

⚠️ 注意事项

  • 转换动画中无法按键
  • 按键有即时反馈
  • 1和4键会立即掉落物品
  • 倾倒后会自动继续掉落循环

🎮 操作示例

  【示例1：从居中调整到左倾1】
    按 2 键一次
  
  【示例2：从居中到左倾2】
    按 2 键两次
  
  【示例3：快速丢弃物品】
    按 1 或 4 键（立即倾倒）
  
  【示例4：从右倾2回到左倾1】
    按 2 键三次
    (右倾2 → 右倾1 → 居中 → 左倾1)

🔧 自定义按键

  在 Unity Inspector 中修改：
  
  SwayController 组件
  └─ 玩家输入
      ├─ Instant Left Topple Key
      ├─ Tilt Left Key
      ├─ Tilt Right Key
      └─ Instant Right Topple Key

✅ 快速记忆

  1 = 左边 + 立即
  2 = 左边 + 逐级
  3 = 右边 + 逐级
  4 = 右边 + 立即

  或者记为：
  1234 = 从左到右
  14 = 快速（立即）
  23 = 慢速（逐级）
