    倾倒阶段回退功能 - 快速参考
═══════════════════════════════════════════════════

📅 更新日期: 2024-10-13
🎮 版本: v1.4.1

───────────────────────────────────────────────────

🎯 核心功能

  在倾倒阶段，按反方向倾斜键可以回退！

───────────────────────────────────────────────────

🎮 操作方法

  【左倾倒阶段】
    按 3键（向右） → 回退到左倾2 ✅
    按 2键（向左） → 无效果 ❌

  【右倾倒阶段】
    按 2键（向左） → 回退到右倾2 ✅
    按 3键（向右） → 无效果 ❌

───────────────────────────────────────────────────

📊 示意图

         按3键回退
    ┌──────────────┐
    │              ↓
  左倾倒 ────────→ 左倾2
    
  
  右倾2 ←──────── 右倾倒
         ↑              │
         └──────────────┘
            按2键回退

───────────────────────────────────────────────────

💡 使用技巧

  【意外倾倒】
    快速按反方向键挽救局面

  【战略丢弃】
    1. 按1或4主动倾倒
    2. 丢弃几个物品
    3. 按反方向键回退
    4. 继续游戏

  【紧急情况】
    倾倒时不要慌张
    记住反方向可以回退

───────────────────────────────────────────────────

⚠️ 注意事项

  • 只能回退到倾2阶段（不是倾1）
  • 同方向按键无效
  • 回退后停止掉落循环
  • 回退后开始5秒倒计时
  • 转换动画中无法回退

───────────────────────────────────────────────────

🎯 记忆口诀

  左倾倒 → 按右键（3）
  右倾倒 → 按左键（2）
  
  反方向 = 回退
  同方向 = 无效

───────────────────────────────────────────────────

✅ 快速测试

  测试步骤：
  1. 按 1键 进入左倾倒
  2. 按 3键 尝试回退
  3. 观察是否回到左倾2
  4. 重复测试右倾倒
