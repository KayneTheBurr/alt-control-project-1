📍 默认起始位置：(-4, -3.4, 3.4)

───────────────────────────────────────────────────

🎯 最底部物品位置

         世界坐标
         X = -4.0    （左侧4单位）
         Y = -3.4    （下方3.4单位）
         Z = 3.4     （靠近相机3.4单位）

───────────────────────────────────────────────────

📦 堆叠示意图

    (以默认间距 1.0 为例)

              ▢  物品5  Y = -3.4 + 4.0 = 0.6
              ▢  物品4  Y = -3.4 + 3.0 = -0.4
              ▢  物品3  Y = -3.4 + 2.0 = -1.4
              ▢  物品2  Y = -3.4 + 1.0 = -2.4
              ▢  物品1  Y = -3.4 + 0.0 = -3.4  ← 起始位置
                        X = -4.0
                        Z = 3.4

───────────────────────────────────────────────────

🔧 Inspector 中调整

  StackController 组件
  └─ 堆叠设置
     └─ Stack Start Position
        ├─ X: -4    （左右）
        ├─ Y: -3.4  （上下）
        └─ Z: 3.4   （深度）

───────────────────────────────────────────────────

💡 快速调整

  【向右移动】增加 X 值
    X = -4 → X = 0 → X = 4

  【向上移动】增加 Y 值
    Y = -3.4 → Y = 0 → Y = 3.4

  【远离相机】减小 Z 值
    Z = 3.4 → Z = 0 → Z = -3.4

───────────────────────────────────────────────────

✅ 测试方法

  1. 运行游戏
  2. 查看 Console 日志
  3. 确认 "StackRoot 本地位置: (-4.0, -3.4, 3.4)"
  4. 在 Scene 视图检查物品位置
